About

This web application offers an advanced lossless compression solution by combining Huffman coding and LZW (Lempel-Ziv-Welch) algorithms, enhanced with Retinex image processing to improve visual quality of compressed images. Designed to efficiently reduce file sizes without sacrificing data integrity, the tool supports both text file compression and image compression workflows.

For images, the app first compresses using LZW and Huffman coding to minimize storage size, then applies Retinex enhancement to restore and improve color contrast and brightness, ensuring visually appealing results even after compression. For text files, the combined compression techniques ensure optimal size reduction with fast encoding and decoding.

This project aims to provide users with a straightforward and effective way to compress files while maintaining quality and usability — ideal for developers, researchers, or anyone looking to optimize storage and transmission efficiency.
